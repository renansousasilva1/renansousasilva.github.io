Sobre Nós

Este projeto foi criado como um exemplo simples de como estruturar múltiplas páginas dentro de um site hospedado no GitHub Pages. Aqui você pode experimentar navegação, estilos básicos e organização de conteúdo sem precisar de back-end ou frameworks complexos.

Você pode ampliar esse template para algo mais elaborado, como blogs, portfólios, páginas institucionais, ou até o MVP do seu think tank com visual limpo e moderno.
